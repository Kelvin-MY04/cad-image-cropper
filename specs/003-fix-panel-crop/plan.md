# Implementation Plan: Fix Panel Detection and Cropping

**Branch**: `003-fix-panel-crop` | **Date**: 2026-03-05 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/003-fix-panel-crop/spec.md`

## Summary

The `ClassicalBorderDetector` selects the most frequently occurring vertical x-position across the whole image. In dense CAD floor plans this picks interior wall lines, not the title block separator. The fix replaces the "most votes" strategy with a height-span aggregation + right-zone search: collect all detected vertical segments, compute their combined y-coverage per x-bin, filter for bins that span ≥ 70% of image height, then return the leftmost qualifying bin found within the right 70–95% of image width.

## Phase 0: Research

*Complete. See [research.md](research.md) for full findings.*

Key decisions:

1. **Root cause**: `_cluster_and_select_dominant_x` uses "most votes" across the full image → interior wall lines win over the separator.
2. **Fix**: Replace dominant-x selection with: aggregate y-spans per x-bin → filter for ≥ 70% height coverage → search right zone (70–95% of width) → return leftmost qualifying bin.
3. **Test updates**: Synthetic stripe must move from x=600 (50%) to x=900 (75%) to be in the right zone.
4. **No contracts needed**: CLI and public API signatures unchanged.

### 2. ClassicalBorderDetector — Method Inventory

#### Methods removed
| Method | Reason |
|---|---|
| `_cluster_and_select_dominant_x` | Replaced by `_aggregate_x_bins` + `_select_separator_x` |

#### Methods modified
| Method | Change |
|---|---|
| `_filter_vertical_lines(lines)` | Return type: `list[int]` → `list[tuple[int, int, int]]`; each tuple is `(x_mid, y_top, y_bottom)` |
| `detect_border(...)` | Updated to pass segments to new helpers; call chain: filter → aggregate → select |

#### Methods added
| Method | Signature | Responsibility |
|---|---|---|
| `_aggregate_x_bins` | `(segments: list[tuple[int, int, int]]) -> dict[int, list[tuple[int, int]]]` | Group `(y_top, y_bot)` spans by x-bin key (`x_mid // BIN_WIDTH`) |
| `_compute_y_coverage` | `(spans: list[tuple[int, int]], image_height: int) -> float` | Merge overlapping y-spans and return coverage ratio |
| `_select_separator_x` | `(bins: dict[int, list[tuple[int, int]]], image_width: int, image_height: int) -> int \| None` | Filter bins to right zone + height threshold; return leftmost qualifying x |

#### `detect_border` call chain (updated)

```text
gray       = _convert_to_grayscale_array(image_array)
morphed    = _apply_vertical_morphology(gray, metadata.height)
thresh     = _apply_binary_threshold(morphed)
lines      = _run_hough_lines(thresh, metadata.height)          # None → DetectionMethod.NONE
segments   = _filter_vertical_lines(lines)                       # list[tuple[int,int,int]]
           → empty → DetectionMethod.NONE
bins       = _aggregate_x_bins(segments)                         # dict[bin_key → spans]
x          = _select_separator_x(bins, metadata.width, metadata.height)
           → None → DetectionMethod.NONE
return DetectionResult(x_coordinate=x, confidence=None, method=DetectionMethod.CLASSICAL)
```
